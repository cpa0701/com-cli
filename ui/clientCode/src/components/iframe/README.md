### iframe

#### API

|  参数  |           说明            |   类型   | 默认值  | 必填 | 版本  |
| :----: | :-----------------------: | :------: | :-----: | :--: | :---: |
|  src   | iframe 中显示的文档的 URL | `String` |    -    |  是  | 0.1.0 |
| width  |        iframe 宽度        | `String` | `100%`  |  -   | 0.1.0 |
| height |        iframe 高度        | `String` | `100vh` |  -   | 0.1.0 |

#### Event

| 事件名 | 说明                    | 参数 | 版本  |
| ------ | ----------------------- | ---- | ----- |
| load   | iframe 加载完成事件回调 | -    | 0.1.0 |
